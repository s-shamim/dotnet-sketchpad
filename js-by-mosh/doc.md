
# Introduction to JavaScript

>

## Summary
- JavaScript is a programming language that conforms to the ECMAScript specification. 
- It runs in browsers via built-in JS engines, and also outside the browser thanks to Node.js. 
- The language has evolved significantly since its first version in 1997, with annual releases starting in 2015.

## Key Concepts
- **JavaScript Engine**: Every browser ships with a JS engine to execute JavaScript code (e.g. SpiderMonkey in Firefox, V8 in Chrome)
- **Node.js**: Created by Ryan Dahl, who took V8 (Chrome's open-source JS engine) and embedded it in a C++ program, allowing JS to run outside the browser
- **ECMAScript**: The official specification that defines the language standard; JavaScript is the programming language that implements it
- **ES6 / ES2015**: A major version of ECMAScript released in 2015 that introduced many modern JS features; annual releases have continued since then

## Common Mistakes / Gotchas
- JavaScript and ECMAScript are not the same thing — ECMAScript is the *spec*, JavaScript is the *language* that conforms to it
- Node.js is not a programming language or a framework — it's a runtime environment that lets JS run on the server/outside the browser
